## Clone the repository

```sh
git clone https://github.com/ps-ector/ector-docker-startup.git
```

## Copy .env.dist to .env

```sh
cp .env.dist .env
```

## Start the project

```sh
make compose
```

For more informations follow the [official documentation](https://docs.ector.store).
